## How to Use These Skills

### With Claude Code (Anthropic)
Place the skill files from `skills/` into `~/.claude/commands/` and invoke them with slash commands:
```
/solve [complex problem description]
/opportunity-scan [geography and sector]
/financial-model [programme or business name]
/gov-proposal [policy and context]
```

### With Other AI Platforms (ChatGPT, Gemini, etc.)
1. Open the relevant `.md` file from the `skills/` folder
2. Copy the full contents
3. Paste it as a system prompt or first message
4. Replace `$ARGUMENTS` with your specific input
5. The AI will follow the skill's structure and produce the output

### Marketing Showcase
Open `sovereign-framework-showcase.html` in any browser. Password: `sovereign2026` (changeable in the file).
